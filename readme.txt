== Description ==

Allows to receive e-mail messages in character sets different from the blog charset.

Based on the original plugin by Anton Skorobogatov, the founder of Russian WordPress community.

== Installation ==

1. Upload `wp-mail-cyrillic` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Select the desired charset in 'E-mail Charset' item under the 'Options' menu.
